| -------- | ----------------- |

---

## ❌ Block AI For

* Vault (passwords)
* Finance
* Credentials

---

## 🧠 AI Flow

```text
User Note → Check sensitivity
        ↓
If safe → Send to AI API
If sensitive → Skip AI
```

---

## 🛡️ Safety Filter (Important)

Before sending to AI:

```js
function isSensitive(text) {
  return text.includes("password") ||
         text.includes("bank") ||
         text.includes("upi");
}
```

👉 If true → DO NOT send

---
